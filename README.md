# 교통사고 분석 및 예측

🟦 데이터 구조
| 데이터    | 설명                 | 변수명            |
| ------ | ------------------ | -------------- |
| 사고 유형별 | 추돌·측면·전도 등 사고 형태   | `df_type`      |
| 위반 유형별 | 신호위반·과속·졸음 등 원인/위반 | `df_violation` |
| 연도별    | 연도별 전체 사고 추세       | `df_year`      |
| 기상 상태별 | 맑음·눈·비 등 기상 영향     | `df_weather`   |
| 연령대별   | 10대~70대 사고 건수      | `df_age`       |
| 운전 경력별 | 초보~베테랑 경력별 사고율     | `df_career`    |
| 지역별    | 시군구/도로별 사고 건수      | `df_region`    |
