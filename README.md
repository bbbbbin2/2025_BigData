# 🚗 교통사고 데이터 분석 & 향후 사고량 예측 프로젝트

본 프로젝트는 **TMACS 교통안전정보관리시스템** 및 **한국도로교통공단**에서 제공하는

연령대·사고유형·기상상태·위반유형·운전경력·지역별 교통사고 데이터를 수집 및 전처리하여

사고 발생 요인을 분석하고, 미래 사고 위험을 예측하는 데이터 분석 프로젝트입니다.

## 📈 5. 주요 분석 결과 (예시)

### 🔹 1) 연령대별 사고 특징

- 20~30대: 과속·주의산만 사고 비율 높음
- 60대 이상: 판단 미숙·운전 미숙 비중 ↑

### 🔹 2) 위반 유형별 영향력

- 신호위반은 교차로 사고와 강한 상관
- 과속은 치명적 사고(사망률 증가)와 직접적 관련

### 🔹 3) 기상 상태 영향

- 비·눈이 올 때 사고 발생 확률 증가
- 빗길은 미끄러짐·추돌 사고 비율 급증

### 🔹 4) 지역별 사고 패턴

- 대도시는 교차로 사고 중심
- 농촌 지역은 야간·보행자 사고 비중 증가

## 🔮 6. 예측 모델 설계

본 프로젝트에서는 다음 3가지 예측 모델 중 1~2개를 중심으로 구현할 수 있습니다.

---

### **예측 모델 1 — 연도별 교통사고 발생량 예측 (시계열 모델)**

- **Model:** ARIMA / Prophet / LSTM
- **Input:** 과거 연도별 사고 건수
- **Output:** 향후 5년 사고 건수 예측

➡ 사고 감소 정책 성과 평가 가능

➡ 발표 효과가 매우 높음

---

### **예측 모델 2 — 화물차 사고 확률 예측**

- Model: RandomForest / XGBoost

- Input: 화물차 중량(톤수), 운행 거리, 운행 시간대, 도로 유형, 기상 상태 등

- Output: "화물차 사고 발생 확률(0~1)"

➡ 화물차 운행 조건 변화에 따른 사고 위험도 자동 산출 가능

➡ 물류 회사·고속도로 공사·지자체가 위험 구간 사전 관리에 활용도 높음

---

### **예측 모델 3 — 지역별 사고 위험 등급 분류 (Classification)**

- **Model:** Logistic Regression / SVM / KMeans
- **Input:** 인구수, 사고건수, 위반율 등
- **Output:** 그래프

➡ 사고 다발지역 선별

➡ 실제 행정기관 방식과 유사
